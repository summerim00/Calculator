계산기 애플리케이션 개요:

1. **HTML 구조**
   - HTML 구조는 입력을 위한 디스플레이 필드와 다양한 연산 버튼을 포함한 계산기 인터페이스로 구성됩니다.
   - 이전 계산 결과를 표시하는 히스토리 섹션과 모든 과거 계산을 삭제할 수 있는 "Clear History" 버튼도 포함되어 있습니다.

2. **CSS 스타일링**
   - CSS는 계산기와 히스토리 섹션을 스타일링하여 사용자 친화적인 인터페이스를 만듭니다.
   - `body`는 화면 중앙 정렬을 위해 스타일링되었으며, 계산기는 깔끔하고 둥근 모서리의 컨테이너와 그림자 효과를 가집니다.
   - 버튼은 사용자가 상호작용할 때 시각적 피드백을 제공하도록 호버 및 클릭 상태 스타일이 설정되어 있습니다.

3. **JavaScript 기능**
   - JavaScript는 버튼 클릭과 키보드 입력 처리를 포함한 계산기 로직을 구현하는 데 사용됩니다.
   - 숫자 추가, 연산자 추가, 디스플레이 초기화, 마지막 입력 삭제, 부호 전환, 결과 계산, 히스토리 추가를 위한 함수들이 제공됩니다.
   - 삼각 함수 계산, 쌍곡선 함수, 제곱근, 팩토리얼 등의 특수 연산을 위한 추가 함수들도 있습니다.
   - 결과를 표시할 때 애니메이션 효과도 제공됩니다.

4. **사용자 상호작용**
   - 사용자는 버튼을 클릭하거나 키보드로 숫자와 연산자를 입력하여 계산기와 상호작용할 수 있습니다.
   - `=` 버튼을 누르거나 `Enter` 키를 사용할 때 자동으로 결과가 계산됩니다.
   - 계산기는 모든 계산 내역을 저장하며, "Clear History" 버튼을 사용하여 기록을 삭제할 수 있습니다.

5. **특수 기능**
   - 계산기는 기본 산술 연산, 제곱 계산, 삼각 함수(`sin`, `cos`, `tan`), 쌍곡선 함수(`sinh`, `cosh`, `tanh`), 제곱근, 팩토리얼을 지원합니다.
   - 입력 필드는 현재 작성 중인 수식을 동적으로 업데이트하며, 사용자는 삭제 버튼을 사용하여 쉽게 실수를 수정할 수 있습니다.
